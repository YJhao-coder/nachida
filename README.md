# 小草神纳西妲表白墙

# 开源协议
本项目采用自写的中文开源协议：[查看开源协议](https://gitee.com/ibaizhan/nahidalove/blob/master/LICENSE)

#### 介绍
原神小草神纳西妲表白墙，基于樱花表白墙进行二次开发

#### 开发环境

开发时PHP版本：7.4.33
开发时MySQL版本：5.6.50

#### 默认信息

安装完成后台登录，后台目录：你的源码/admin
默认账号：admin 
默认密码：123456

#### 数据库相关

导入数据库文件：nahida.sql
数据库连接配置文件：/nahida_bz/mysql.php

#### 相关说明：本源码虽然基于作者樱振宇的樱花表白墙进行修改开发并二次开源，但本源码的数据库和樱花表白墙的数据库文件并不完全通用，此源码使用樱花表白墙的数据库文件可能会存在BUG
